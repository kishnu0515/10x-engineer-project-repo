## Project Overview and Purpose

PromptLab is a platform designed for AI engineers to efficiently **store, organize, and manage their AI prompts**. The tool functions similarly to "Postman for Prompts," where users can:
- 📝 Store prompt templates with variables (`{{input}}`, `{{context}}`)
- 📁 Organize prompts into collections
- 🏷️ Tag and search prompts
- 📜 Track version history
- 🧪 Test prompts with sample inputs

Our aim is to streamline the process of prompt engineering, helping teams to collaborate, experiment, and improve their AI models with ease.

## Features

- Store and manage prompt templates.
- Organize prompts into collections.
- Add and filter prompts by tags.
- Version tracking for prompts.
- Comprehensive API documentation.

## API Endpoint Summary with Examples

### Health Check
- **GET** `/health`: Returns the health status of the API.

### Prompts
- **GET** `/prompts`: Retrieves a list of prompts.
- **POST** `/prompts`: Creates a new prompt.

### Collections
- **GET** `/collections`: Retrieves a list of collections.
- **POST** `/collections`: Creates a new collection.

Replace these examples with more as you build your application.
